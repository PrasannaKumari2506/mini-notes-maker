# Mini Notes Maker 📝

A lightweight, responsive web application designed for creating, organizing, and managing daily notes in real-time. Built as a serverless frontend application powered by Firebase.

## 🚀 Features

- **Real-Time Database:** Notes are instantly synchronized and saved securely using Firebase Firestore.
- **Full CRUD Functionality:** Create, read, update, and delete notes effortlessly.
- **Dynamic UI:** A clean, minimal, and responsive user interface that adapts to mobile, tablet, and desktop screens.
- **Instant Updates:** No manual page refreshes required to see database changes.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend-as-a-Service (BaaS):** Firebase (Firestore Database)

## 📂 Project Structure


## 📦 Getting Started & Installation

Follow these steps to get a local copy of the project running on your machine:

### 1. Clone the Repository

```bash
git clone [https://github.com/PrasannaKumari2506/mini-notes-maker.git](https://github.com/PrasannaKumari2506/mini-notes-maker.git)
cd mini-notes-maker

```

### 2. Configure Firebase

Because this project utilizes Firebase, you will need to add your own Firebase configuration credentials inside `firebase.js`.

For step-by-step guidance on setting up your Firebase Console and getting these keys, please refer to the [SETUP.md](https://www.google.com/search?q=./SETUP.md) file included in this repository.

### 3. Run the Application

Since this is a client-side application, you don't need to install any heavy backend servers or dependencies:

* Simply double-click `index.html` to open it locally in your web browser.
* Alternatively, run it using a local development server like VS Code's **Live Server** extension for the best development experience.

## 🔒 Security Rules

Database security configurations are maintained in `firestore.rules`. Make sure to deploy these rules to your Firebase console to ensure data security and integrity.
